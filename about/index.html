Seal Report 学习笔记

2019-06-01 :: SoAi
#BI 

Hi there

无意中看见一款自助式 BI， C# 开发 而且是开源、免费。

安装包只有15兆，.NET 库 另算。但运行起来已经是飞快的了；渲染方面，默认只配给你HTML，其他的都是商业产品，没法配。

可以支持PDF EXCEL 等格式。感觉极好！

所以我决定把学习的过程记录下来。

该软件给我的初步感觉，上手比 tableau 难很多，那个东西我不看指南都能搞定。 如果你不介意用破解软件，那么tableau真的很好。

Seal Report 不是一个面向办公室人员的软件，对新手并不友好；它更加适合开发人员，尤其对数据库领域有比较高的要求。

然后让熟悉业务的人来给你提要求，这样才能做出高质量的报表，否则还不如用 EXCEL

最后，这个东西还支持定时器，可以让你稳定更新 HTML 文件； 那么，配合web server，可以实现定时更新并分享。

以上就是我的初步感受。
