第一章 下载和安装

2019-01-25 :: So Ai
#BI 

1

我们去 https://sealreport.org 下载即可，自己去找 DownLoad。 分 32位和64位 ，因为是 C# ，所以只有windows平台, 同时还提供源码下载。

下载后，双击即可完成安装，非常的给力。

2

安装后出现两个图标

  • Seal Server Manager
    用于管理数据源

  • Seal Report Designer
    用于设计报表

安装时建议顺带安装演示数据库和演示报表，方便你学习，自带的数据库是大家熟悉的 Northwind.mdb

支持windows自身的大部分链接方式 比如这里用的是很古老的 ODBC JET 4.0

同时也支持文本格式的数据，范例里面自带一个读取RSS的筛选器。刚开始的时候还以为能够支持 no sql 数据库这么吊，现在看来是不行的.
